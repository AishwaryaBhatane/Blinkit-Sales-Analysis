# Blinkit Sales Analysis Dashboard 📊

## 📌 Project Overview

This project presents an interactive Power BI dashboard analyzing sales and operational metrics for Blinkit – India's Last Minute App.
The dashboard helps uncover insights related to sales performance, product categories, outlet locations, outlet sizes, and customer ratings.

The goal of this project is to demonstrate data analysis, data visualization, and business insight generation using Power BI.

# 🎯 Business Objectives

The dashboard answers key business questions such as:

* What is the total sales generated across all outlets?
* Which item types generate the highest sales?
* How does fat content affect product sales?
* Which outlet size and outlet location contribute the most revenue?
* What is the average rating and average sales per outlet?
